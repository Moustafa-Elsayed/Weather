<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone https://github.com/Moustafa-Elsayed/Weather.git
```

<p>2. Install npm modules</p>

```
yarn install
```


<p>5. Check production build locally using below command</p>

```
 yarn android 
```
